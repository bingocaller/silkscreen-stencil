Now, this here's a <slk-button> (pronounced "silk button"):

Primary

It's a Web Component that uses Shadow DOM so it is entirely independent from the rest of the DOM, which means all styles are automatically scoped to the component, i.e. encapsulated.

Secondary

Gost

  • Role: ghost
